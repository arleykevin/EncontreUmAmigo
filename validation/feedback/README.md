# Feedback e Ajustes

Registro de melhorias implementadas com base no feedback durante o desenvolvimento.

## Ciclo 1: Interface
* **Feedback:** A logo antiga não remetia à identidade visual.
* **Ação:** Criação e implementação da nova logo (Patinha SVG) no site e no ícone do App.

## Ciclo 2: Funcionalidade Mobile
* **Feedback:** O formulário de cadastro estava quebrado em telas pequenas (modo escuro/padding excessivo).
* **Ação:** Refatoração do componente `RegisterForm` para responsividade mobile e fixação de fundo branco.

## Ciclo 3: Regras de Negócio
* **Feedback:** Necessidade de gerenciar solicitações de adoção e não apenas excluir.
* **Ação:** Implementação de status (Analisando/Aprovado/Reprovado) e etiquetas visuais.
